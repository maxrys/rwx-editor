
# Политика конфиденциальности

Исходные коды приложения открыты и доступны в репозитории для ознакомления
и анализа по адресу "https://github.com/maxrys/rwx-editor".


## Структура приложения

Приложение является связкой из двух частей:
- главное приложение `RWX Editor`;
- Finder-расширение `RWX Editor Extension`.

Главное приложение `RWX Editor` состоит из:
- главного окна `RWX Editor | Settings` (в нём пользователь определяет
  какие директории будут доступны расширению `RWX Editor Extension` для работы
  с файлами/директориями через контекстное меню Finder);
- динамические окна `RWX Editor`, которые будут создаваться
  в контексте каждого файла/директории (после клика в контекстном меню
  Finder по пункту `RWX Editor`).

Расширение `RWX Editor Extension` показывает дополнительный пункт меню `RWX Editor`
в контекстном меню Finder. При клике на данный пункт меню происходит запуск
основного приложения и динамическое создание окна `RWX Editor` в контексте этого
файла/директории.

Запуск расширения `RWX Editor Extension` возможен лишь после того, как
пользователь включит соответствующий переключатель в настройках системы.


## Модификация файловой системы

Приложение изменяет POSIX-права на файлах и директориях по требованию
пользователя в рамках возможностей данного приложения.

Изменение прав вне разрешённых директорий не доступно.

Изменение прав на файлы и директории, не принадлежащие пользователю,
не доступно.


## Хранение данных

Приложение использует стандартный механизм `Core Data` для хранения
"списка закладок" на директории в локальной базе данных.


## Другие активности

Приложение:
- работает в изолированной среде ОС (песочнице);
- не регистрирует нажатие пользователя;
- не получает доступ к буферу обмена;
- не отправляет и не принимает данные по сети;
- не собирает и не хранит персональные данные;
- не передаёт данные третьей стороне.
