## Description

This Finder extension (open source) creates a new menu item "RWX Editor" in the Finder context menu.
After clicking on a menu item (in the context of a file or directory), the "RWX Editor" window is called up, in which the user can:
- view information about the file or directory (creation time, size, etc.);
- edit access rights to the file or directory;
- edit the owner of the file or directory;
- edit the group.
